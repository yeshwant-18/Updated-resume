Hidden Markov Model based trust evaluation in VANETs (TCL)

This project developed a trustworthy model using Hidden Markov Models (HMM) and Blockchain technology to enhance accuracy in wireless communication within Vehicular Ad-hoc Networks (VANETs).

PSO, ABC, AND A2BC BASED FEATURE SELECTION FOR CLASSIFICATION MODELS IN AN EDUCATION APPLICATION

This project is a comparison of Swarm Intelligence-based optimization algorithms, like Particle Swarm Optimization (PSO), Artificial Bee Colony (ABC), and Advanced Artificial Bee Colony(A2BC). The comparison is done based on the accuracy they provide when used for feature selection to classify student success. The accuracy is calculated using SVC, Random Forest, XGBoost, and Voting classifiers. The Advanced Artificial Bee Colony algorithm is an improved version of ABC that includes extensions to the ABC algorithm.

Spam detection in Twitter

This project created a user-friendly application aimed at detecting spam on Twitter and calculating the percentage of fake tweets. It showcases the application of machine learning in enhancing social media security and user experience.

Quality of wine from the dataset related to red variant of the Portuguese "Vinho Verde" wine

Using Linear Regression and the sklearn library, this project analyzed a dataset to predict the quality of wine, demonstrating the practical use of machine learning in food and beverage quality assessment.
